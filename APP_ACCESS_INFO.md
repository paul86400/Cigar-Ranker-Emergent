# Cigar Ranker - Access Information

## Web Preview (Recommended)
The app is fully functional via web browser on any device:

**URL:** https://stogie-rater.preview.emergentagent.com

**Access from Phone:**
1. Open your phone's web browser (Safari, Chrome, etc.)
2. Navigate to the URL above
3. The app is fully responsive and works on mobile browsers

## Expo Go Access
The Expo development server is running with tunnel mode, but connecting via Expo Go may have limitations in the preview environment.

**Current Configuration:**
- Tunnel Subdomain: stogie-rater
- Metro Port: 3000
- Tunnel Status: Active

**If you want to try Expo Go:**
The preview environment is primarily designed for web preview. For full Expo Go support, you would need to:
- Run the app in a local development environment
- Or deploy to Expo's EAS services
